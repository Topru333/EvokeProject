# EVOKE PROJECT

## Project setup

First of all, lets download modules.Open terminal and go to (cd) ReactProject folder. Run 'npm i'.
After we need to build bundle.js file. . Run 'npm run webpack' command. Now bundle.js in ReactProject/dist folder.

## Database
Open in visual studio solution explorer (обозреватель серверов). Click 'Connect to database'. Choice microsoft sql sserver file and file in Company.mdf in App_Data.
Click connect.
